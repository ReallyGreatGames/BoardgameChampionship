## Design Context

### Users
Serious, competitive board game players at national and international championships. These are not casual fans — they are invested, skilled, and present at high-stakes events. Players use the app on their own devices during a tournament. Admins (tournament organizers) use the admin view on one or more shared tablets and/or a browser at the event venue. The interface must be sharp enough for competitors who care about polish, fast enough for event-day pressure, and readable on tablets in variable lighting.

### Brand Personality
**Clean, competitive, fun.**
The app should feel like a premium esports tournament platform that happens to be for board games — not a generic CRUD app. Energy and personality without chaos. Serious but not sterile. The current flat-black look is too austere; the design should breathe with the color palette and feel alive.

### Color Tokens

**Dark Mode**
| Token      | Value     |
|------------|-----------|
| text       | `#daecfa` |
| background | `#010509` |
| primary    | `#81b8ed` |
| secondary  | `#8c1591` |
| accent     | `#e02883` |

**Light Mode**
| Token      | Value     |
|------------|-----------|
| text       | `#051624` |
| background | `#f6fafe` |
| primary    | `#12497d` |
| secondary  | `#e66cea` |
| accent     | `#d61f7a` |

**Usage guidance:**
- `primary` — interactive elements, links, focused states, key UI chrome
- `secondary` — supporting highlights, tags, secondary actions
- `accent` — CTAs, high-emphasis moments, badges, competitive highlights
- `text` — body copy, labels
- `background` — screen base (NOT pure black / pure white — use these tokens)

### Aesthetic Direction
- **Dark-first** but light mode is a first-class citizen — both palettes should be fully implemented
- Deep navy-black background (`#010509`) over pure black — gives depth without looking washed out
- Bold use of the accent pink/magenta (`#e02883` / `#d61f7a`) for competition moments — winners, scores, key actions
- The cool-blue primary (`#81b8ed`) carries most interactive UI chrome
- Purple secondary (`#8c1591` / `#e66cea`) adds personality and differentiates supporting elements
- Typography should feel confident — generous weight contrast between headings and body
- Cards and containers: subtle elevation via border/bg layering, not heavy shadows
- Avoid: pure `#000000` or `#ffffff` backgrounds, flat gray monotone, enterprise/admin-bland aesthetics

### Design Principles

1. **Competitive clarity** — At a live event, the player's next action must be obvious in under a second. Hierarchy is non-negotiable.
2. **Color with purpose** — Every use of accent/secondary signals something: progress, status, selection, urgency. No decorative color splashes.
3. **Both modes, equal quality** — Light and dark mode are both used (tablets in bright venues vs. phones in dim rooms). Neither is an afterthought.
4. **Confidence over caution** — Bold type, strong contrast, decisive spacing. This is a championship, not a settings screen.
5. **Platform-native feel** — Built with React Native StyleSheet, no CSS-in-JS. Designs should feel native and snappy, not web-ported.
